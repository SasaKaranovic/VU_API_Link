What is API Link?

An application programming interface (API) is a way for two or more computer programs or components to communicate with each other.

By design API will only respond to a requst and it will never initiate a request.
So let's say for example that we want to pull some information from a remote API and show it on our VU dials.

The remote API will wait for a request before it responds with data.
Also the VU Server API will wait for a request before it updates VU dials.
But neither of them will make the first request... so it's a stalemate.

Enter API link!
This is a very simple application that allows you to create a link between remote API and VU Server API.
It does this by allowing you to create a 'link' file. This file describes which API/URL to poll, how often, and what is the data that we want passed to VU Server.

Also the remote API might provide data that is not directly consumeable by the VU server.
To address this API Link allows you to modify the data before passing it to VU Server.
This is done by using 'modifiers'.
As the name implies, modifiers allow you to modify the value before it's sent to VU Server.
